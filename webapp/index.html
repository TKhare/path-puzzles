A set of path puzzles: connect each pair of matching letters with a path, covering as many empty squares as you can. Solve one yourself, or run a model and watch it try.

  • Connect each letter to its twin with an orthogonal path (up / down / left / right).
  • Intermediate paths may cross each other (a single empty square can be used by more than one letter's path), but may not pass through any other letter.
  • Minimize the number of empty squares.
