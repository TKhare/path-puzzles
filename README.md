# path puzzles

Connect each pair of matching letters with a path, minimizing empty squares (paths may overlap).
A static showcase of small puzzles that frontier models (Opus 4.5 / Sonnet 4.6 / Opus 4.8 at low·high·max
effort) get wrong — play them yourself and see measured per-model pass rates in the Stats tab.

Static build (no backend); the live "run a model" feature is disabled here.
